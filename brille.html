Brille brand identity design is the final assignment project for Graphic Design Specialization course, and Brille itself is a fictional startup that was purposely made for the project. This company develops smart glasses powered with a powerful, JARVIS inspired A.I. system.

The goal is to create a memorable name and an identity showcasing the company's philosophy in creating advanced, convenience, and fashion-able products. The company's name, 'Brille' is derived from Germany word meaning eyes. The whole identity is designed to bring out smartness as a focus, while still being clean and sleek to signify the company's philosophy.

The logo is redesigned with a new bold wordmark that shouts out the organization's name and primary mission. The name 'Brille' is a Germany term meaning glasses. It's straightforward to the company's main idea, and it's easy to pronounce and remember.

The final logotype features the capita-lised 'Brille' word, with a modified version of I and L letter. The logomark is taken directly the modified letters. The logomark carries the three main philosophies of the brand, which is advanced, companionable and convenient. The selected colour palette is inspired by those.

The brand's modular system utilises the shape of the logomark to create a pattern inspired by brain structures and the matrix shapes. The designed pattern can be used to be a complementing element for the branding layout.
